vkBeautify SQL parser covers only SELECT statement as most javascript developers most time have deal with SELECT. As SQL syntax itself is very large and complex, some complex statements can be formatted not as perfect as you'd make it manually. But anyway, this tool helps you to save time on reading unformatted SQL statement.

Click on "Pretty Print" button to beatify the text.

Click on "Minify" button to minify the text (not sure if anybody needs it, but I keep it for consistency :)

"Preserve Comments" is not implemented for SQL part or the application.
