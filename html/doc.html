Documentation

  • No configuration is needed
  • No third party libraries are involved
  • Include in the page head: <script type="text/javascript" src="vkbeautify.js"></script>
  • Usage:
    • Pretty print: vkbeautify.xml|json|css(text);
    • Minify: vkbeautify.xmlmin|jsonmin|cssmin(text [,preserveComments]);
  • Parameters:
    • @text - String; XML, JSON or CSS text to beatufy;
    • @preserveComments - Bool; [optional] flag, which is used in minxml and mincss functions only.
      Set this flag to true to prevent removing comments from @text;
    • Return - String
  • Usage Examples:
    • vkbeautify.xml(text); // pretty print XML
    • vkbeautify.json(text); // pretty print JSON
    • vkbeautify.css(text); // pretty print CSS

    • vkbeautify.xmlmin(text [, true]);// minify XML
    • vkbeautify.jsonmin(text);// minify JSON
    • vkbeautify.cssmin(text[, true]);// minify CSS
