Documentation

  • No configuration is needed
  • No third party libraries are involved
  • Include in the page head: <script type="text/javascript" src="vkbeautify.js"></script>
  • Usage format: vkbeautify(text, format [,preserveWS] );
  • Parameters:
    • @text - String; XML or JSON text to beatufy;
    • @format - String; can be either 'xml' or 'json';
    • @preserveWS - Bool; [optional] flag, which instruct application do not remove extra white spaces( which is done by default)

      Example:
      original text: <elm>       </elm>
      preserveWS is undefined or false: <elm> </elm>
      preserveWS is true: <elm>       </elm>


  • Usage Examples:
    • vkbeautify(text, 'xml');
    • vkbeautify(text, 'json');
    • vkbeautify(text, 'xml', true);//Preserve whitespaces
    • vkbeautify(text, 'xml', true);//Preserve whitespaces
