Overview

vkBeautify is a small, simple and powerfull javascript plugin to beatify XML or JSON text.

Consider to use it if you:

  • create XML or JSON text manually at client side
  • create XML or JSON with tool, but let user to check or edit the result manually
  • get XML or JSON data from web service and check or edit it before or after processing the data
  • other cases when you need to create, edit or check content of an XML or JSON file.

This plugin:

  1. less then 1.5k if minified
  2. has only one function
  3. takes less then 5 mSec to process 50K text
    ( Celeron 2.2 GHz, 2.0GB, any of 3 major browsers)
